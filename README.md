#JAVA CMS Template#

## Feature:
* Admin Page.
* Employee Management.

## Technology Stack:
* Apache Tomcat.
* Spring Framework.
* Spring Security.
* Hibernate ORM.
* MySQL Database.

## Step to Build:
* Import as Maven Project.
* Modify and Fix Your classpath.
* Edit username and password for Your Database in hibernate.cfg.xml.
* Export application.sql File to Your MySQL Database Schema.
* Run as Maven Project.
